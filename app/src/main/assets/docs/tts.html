tts - 文本转语音#

tts模块提供了文本转语音（Text-to-Speech）功能，可以在设备上朗读指定的文本。

tts.speak(text[, options])#

  • text <string> 要朗读的文本
  • options <Object> 选项
    • pitch <number> 音调 (0.0-2.0, 默认1.0)
    • speechRate <number> 语速 (0.0-2.0, 默认1.0)
    • volume <number> 音量 (0.0-1.0, 默认1.0)

朗读指定的文本。这会将当前朗读队列中的其他文本清除，然后朗读新的文本。

// 基本用法
tts.speak("你好，Auto.js");

// 使用选项
tts.speak("这是一段较慢的语音", {
    speechRate: 0.8,  // 语速较慢
    pitch: 1.1        // 音调稍高
});

tts.speakAsync(text[, options], callback)#

  • text <string> 要朗读的文本
  • options <Object> 选项
    • pitch <number> 音调 (0.0-2.0, 默认1.0)
    • speechRate <number> 语速 (0.0-2.0, 默认1.0)
    • volume <number> 音量 (0.0-1.0, 默认1.0)
  • callback <function> 朗读完成后的回调函数，参数为error

异步朗读指定的文本。朗读完成后调用回调函数。

tts.speakAsync("这是一段异步朗读的文本", function(error) {
    if (error) {
        console.error("朗读失败: " + error.message);
    } else {
        console.log("朗读完成");
    }
});

tts.preview(text)#

  • text <string> 要预览朗读的文本

预览朗读文本，通常用于短文本的快速朗读。

tts.preview("短文本");

tts.setLanguage(language)#

  • language <string> 语言代码，如 'zh-CN', 'en-US', 'ja-JP' 等
  • 返回 <boolean> 是否设置成功

设置TTS引擎的语言。这会影响后续朗读的语音语言。

// 设置为中文
tts.setLanguage('zh-CN');

// 设置为英文
tts.setLanguage('en-US');

tts.getLanguage()#

  • 返回 <Object> 当前语言设置

获取当前TTS引擎的语言设置。

var currentLanguage = tts.getLanguage();

tts.getAvailableLanguages()#

  • 返回 <Array> 支持的语言列表

获取设备支持的TTS语言列表。

var languages = tts.getAvailableLanguages();
console.log("支持的语言数量: " + languages.length);
for (var i = 0; i < Math.min(5, languages.length); i++) {
    console.log("- " + languages[i].displayName + " (" + languages[i].language + ")");
}

tts.setPitch(pitch)#

  • pitch <number> 音调值 (0.0-2.0, 默认1.0)

设置TTS引擎的音调。较高的值会使声音更尖锐，较低的值会使声音更低沉。

tts.setPitch(1.2);  // 设置较高的音调

tts.setSpeechRate(speechRate)#

  • speechRate <number> 语速 (0.0-2.0, 默认1.0)

设置TTS引擎的语速。较高的值会加快语速，较低的值会减慢语速。

tts.setSpeechRate(1.5);  // 设置较快的语速

tts.setVolume(volume)#

  • volume <number> 音量 (0.0-1.0, 默认1.0)

设置TTS引擎的音量。该值会在下次朗读时生效。

tts.setVolume(0.8);  // 设置80%的音量

tts.pause()#

暂停当前的朗读。实际上会停止当前朗读。

tts.pause();  // 停止当前朗读

tts.stop()#

停止当前的朗读。

tts.stop();  // 停止当前朗读

tts.isSpeaking()#

  • 返回 <boolean> 是否正在朗读

检查TTS引擎是否正在朗读文本。

if (tts.isSpeaking()) {
    console.log("正在朗读中...");
} else {
    console.log("没有在朗读");
}

tts.shutdown()#

关闭TTS引擎并释放资源。在脚本结束时会自动调用此方法。

tts.shutdown();  // 关闭TTS引擎
